Nouveaux relais

Afin d’enregistrer un nouveau relais, il faut d’abord sélectionner catégorie, discipline de relais et société. La liste de disciplines comprend dans ce cas uniquement des disciplines de relais.
Les relais ne peuvent toutefois être enregistrés que lorsqu’ils sont aussi enregistrés dans l’horaire.
Après la sélection, un nom de relais librement choisi peut être attribué. Toutefois le même nom ne peut être attribué qu’une seule fois par catégorie et discipline.
Une équipe ne peut être répartie que dans la mesure où cette catégorie est organisée comme concours par équipe (par ex. championnat interclubs) et où l’ équipe a déjà été enregistrée.
En plus une meilleure performance peut être enregistrée dont on peut tenir compte dans l’organisation des séries.

La liste des athlètes comprend tous les athlètes inscrits pour la catégorie choisie, la société choisie et, si disponible, l’équipe choisie dans cette discipline de relais. Les athlètes sont soit répartis dans aucun soit dans ce relais.
La répartition d’un athlète dans un relais se fait par l’attribution d’une position dans le relais (numéroté de manière ascendante). Cette position peut en tout temps être changée ou supprimée.
Une répartition des athlètes n’est pas absolument nécessaire.
