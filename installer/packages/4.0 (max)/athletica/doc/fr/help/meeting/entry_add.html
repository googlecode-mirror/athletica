Nouvelle inscription

Afin d’enregistrer une nouvelle inscription, il faut tout d’abord sélectionner la société sur la liste des sociétés et une catégorie définie.

Le prochain dossard disponible est automatiquement saisi. On peut écrire par-dessus ce dossard. Si ce numéro a déjà été attribué, un avertissement sera affiché lors de l’enregistrement et le prochain numéro disponible sera affiché.

On peut naviguer d’un champ de saisie au prochain avec une touche tabulateur. Nom et prénom sont introduits comme d’habitude. L’année de naissance peut être introduite avec quatre ou deux chiffres.
Pour les doubles licences ou les CoA, la deuxième société est automatiquement affichée.
Une équipe ne peut être répartie que si cette catégorie est organisée comme concours par équipe (par ex. concours de société) et si l’ équipe a déjà été enregistrée.

Les disciplines peuvent également être sélectionnées avec la touche tabulateur. Pression sur la touche espace sélectionne cette discipline.
Par discipline on peut mettre une meilleure performance, dont il sera tenu compte pour la répartition dans les séries.
Si l’athlète est inscrit dans une catégorie qui a été définie comme concours multiple ou concours multiple par équipe, les disciplines sont déjà sélectionnées. Si souhaité il est possible de désélectionner certaines disciplines.

Attention: Seul un athlète peut être enregistré avec le même nom, prénom, année de naissance et société. Cette restriction peut par ex. être contournée avec un ajout au prénom.

Après avoir enregistré avec succès, le masque est supprimé. Société et catégorie restent maintenues. Le tabulateur se trouve dans le champ du nom.
