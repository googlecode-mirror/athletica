Anmeldungen

Die Liste links zeigt alle gemeldeten Athleten. Diese Liste kann nach Startnummer oder Name sortiert werden. Durch einen Mausklick auf einen Athleten werden dessen Details im Hauptteil des Fensters angezeigt.

Befehle

Bestleistungen aktualisieren

Diese Funktion aktualisiert die Bestleistungen (genauer: Meldeleistungen für Bahneinteilung) mit den in den Stammdaten verfügbaren Resultaten. Somit können bereits erfasste (oder via Meeting-Abgleich eingegangene) Anmeldungen und deren Meldeleistungen nachträglich aktualisiert werden. Wird die Option "Stehen lassen" gewählt, so werden von Hand eingetragenen Bestleistungen nicht überschriben.

Startnummern

Im Hauptteil wird ein Formular zur Startnummernverteilung angezeigt.

Drucken

Im Hauptteil wird ein Formular zum Drucken von Anmeldungslisten angezeigt.

Neue Anmeldung

Aufruf einer Eingabemaske zur Erfassung einzelner Anmeldungen.

Suchen

Im Suchfeld kann entweder ein Athletenname oder eine Startnummer eingegeben werden. Durch Drücken der Eingabetaste wird der Suchauftrag abgeschickt und die Ergebnisliste im Hautteil angezeigt. Nach Anklicken eines Athleten in dieser Liste werden dessen spezifische Daten angezeigt.

Anmeldung ändern

Nach Anklicken einer Anmeldung aus der Liste links oder aus der Ergebnisliste nach einem Suchauftrag wird im Hauptteil ein Formular zu Änderung der Athletendaten geöffnet. Ausser Verein können alle Werte geändert werden (der Verein lässt sich nur in der Athletendatenbank ändern).
Beim Ändern der Startnummer wird geprüft, ob diese nicht bereits vergeben ist. Die Kategorie kann erst neu gesetzt werden, wenn alle Disziplinenanmeldungen gelöscht worden sind. Alle Änderungen werden sofort nach verlassen des entsprechenden Eingabefeldes in die Datenbank geschrieben.

Eine Mannschaft kann nur zugeteilt werden, sofern diese Kategorie als Mannschaftswettbewerb (z.B. Vereinsmeisterschaft) durchgeführt wird und die Mannschaft bereits erfasst wurde.
Eine Disziplin kann nur geändert werden, wenn der Athlet nicht bereits einer Wettkampfserie der entsprechenden Disziplin zugeteilt wurde. Wird eine Disziplin markiert erscheint ein neues Formularfeld zu Eingabe einer Bestleistung.

Wurde der Athlet einer Disziplin (vorwiegend Staffel) einer anderen Kategorie zugeordnet, wird dies speziell angezeigt. Diese Disziplinenanmeldung kann ebenfalls gelöscht werden.

Anmeldung löschen

Nach Anklicken einer Anmeldung kann diese auch gelöscht werden. Dies ist nur möglich, wenn der Athlet nicht bereits in einer Wettkampfserie eingeteilt ist. Die allgemeinen Athletendaten bleiben dabei allerdings in der Athletendatenbank stehen.
