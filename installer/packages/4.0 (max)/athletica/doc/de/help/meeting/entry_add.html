Neue Anmeldung

Um eine neue Anmeldung zu erfassen, muss zuerst der Verein aus der Liste der Vereine und eine definierte Kategorie ausgewählt werden.

Die nächste verfügbare Startnummer wird automatisch eingefüllt. Diese Startnummer kann überschrieben werden. Falls diese Nummer bereits vergeben wurde, wird dies beim Speichern mit einer Warnung angezeigt und die nächsthöhere, verfügbare Nummer wird angezeigt.

Mit der Tabulatortaste kann von einem Eingabefeld zum nächsten navigiert werden. Name und Vorname werden wie gewohnt eingesetzt. Der Jahrgang kann sowohl vierstellig als auch zweistellig eingegeben werden.
Bei Doppellizenzen oder LG's wird automatisch der Zweitverein angezeigt.
Eine Mannschaft kann nur zugeteilt werden, sofern diese Kategorie als Mannschaftswettbewerb (z.B. Vereinsmeisterschaft) durchgeführt wird und die Mannschaft bereits erfasst wurde.

Die Disziplinen können ebenfalls mit der Tabulatortaste angewählt werden. Druck auf die Leerschlagtaste selektiert diese Disziplin.
Pro Disziplin kann eine Bestleistung gesetzt werden, welche bei der Serieneinteilung berücksichtigt werden kann.
Wird der Athlet in einer Kategorie angemeldet, welche als Mehrkampf oder Mannschaftsmehrkampf definiert wurde, sind die Disziplinen bereits selektiert. Falls gewünscht können einzelne Disziplinen deselektiert werden.

Achtung: Es kann nur ein Athlet mit identischem Namen, Vornamen, Jahrgang und Verein gespeichert werden. Diese Einschränkung kann z.B. mit einem Vornamenszusatz umgangen werden.

Nach dem erfolgreichen Speichern, wird die Maske gelöscht. Verein und Kategorie bleiben erhalten. Der Tabulator steht im Namensfeld.
