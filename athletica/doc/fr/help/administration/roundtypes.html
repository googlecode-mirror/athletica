Types de tour

Les types de tour servent à désigner les tours de la compétition et des séries. Ils sont utilisés lorsqu’un concours s’étend sur plusieurs tours. (par ex.. éliminatoire -> demi-finale -> finale).

Attention: La suite des tours d’une épreuve est uniquement déterminée par l’heure de début, telle qu’elle a été fixée dans l’horaire. L’utilisateur a donc le choix de démarquer le premier tour en le munissant de l’appellation finale.

Emploi

Un nouveau type de tour est ajouté au moyen du formulaire en haut de la page et doit être enregistré.
Les modifications apportées aux données existantes sont directement enregistrées dans la base des données, lorsqu’on passe à la case suivante.
Un type de tour ne peut être supprimé que lorsqu’il n’est plus référencé.

Données

Type
Abréviation, précède l’appellation de la série (par ex. É1 ou FA).
Nom
Appellation complète, telle qu’elle est utilisée dans les titres.
Évaluation
Définit si le classement est donné pour toutes les séries (commun) ou individuellement par série (séparé).
La troisième variantediscipline standard signifie que l’évaluation est faite selon le type de discipline. Les disciplines sur piste (courtes distances avec et sans vent) sont par défaut évaluées individuellement par série, les disciplines techniques ensemble.
