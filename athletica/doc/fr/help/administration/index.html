Administration

Différents sous-menus vers l’administration des données de base peuvent être appelés par ce menu principal. Des fonctions pour configurer les paramètres d’application ainsi que différent liens secondaires se trouvent sur la page du menu principal.

Configuration

La langue de l’application peut être fixée ici. Elle est enregistrée comme cookie chez chaque utilisateur.

Liens

Sous ce titre se trouvent des renvois vers la documentation ainsi que vers les directives relatives aux licences.

Base des données

Sauvegarder des données

Lors de la sauvegarde les données sont enregistrées dans un fichier SQL sur l’ordinateur de l’utilisateur. L’utilisateur peut choisir librement le nom et l’emplacement d’enregistrement.

Restauration des données

Pour la restauration des données, l’utilisateur doit tout d’abord choisir un fichier SQL. Si la version des données sauvegardées correspond à la version installée, le fichier sera lu.

Attention: Les données existantes seront supprimées! Avant la restauration il est recommandé de procéder préalablement à une sauvegarde des données existantes.

Vider le cache

Avec cette fonction on peut résoudre des problèmes avec des athlètes et/ou des sociétés non trouvés.

Attention: Les athlètes et les sociétés saisis sont supprimés pour que ceux-ci puissent à nouveau être téléchargés par la mise à jour des données de base.
Ceci présuppose que pour les meetings saisis dans Athletica aucun athlète ne soit enregistré.
