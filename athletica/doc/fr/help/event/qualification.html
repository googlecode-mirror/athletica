Qualification

Le nom du tour précédent ainsi que le nombre d‘athlètes qualifiés sont affichés. Les paramètres suivants influencent l’organisation des séries du tour suivant:
Nombre d’athlètes par série
Combien d’athlètes une série peut compter au maximum.
Nombre de couloirs
Si cette valeur est plus élevée que le nombre d’athlètes les couloirs surnuméraires sont laissés vides. Ceci est utile lorsque tous les couloirs ne sont pas occupés, mais que les couloirs surnuméraires doivent tout de même être affichés et imprimés sur les formulaires.

Ce paramètre ne peut être mis que pour les courses.

Mode d’organisation
Lors d’une organisation en fonction de la meilleure performance les athlètes sont répartis dans les séries en fonction de la performance des couloirs centraux vers l’extérieur.

Si l’organisation se fait selon la Règle IWB on tire au sort les couloirs centraux pour les athlètes les plus rapides, les couloirs extérieurs pour les athlètes plus lents.
