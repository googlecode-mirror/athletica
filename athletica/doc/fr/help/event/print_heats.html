Imprimer séries

Cette fonction est à disposition après avoir terminé l’organisation des séries et sert à définir le paramètre d’impression suivant:
Nombre d‘essais
En standard trois essais sont définis.
Seulement le meilleur résultat
Pour les disciplines techniques, il est possible de cocher seulement le meilleur résultat. Comme cela le curseur saute directement sur la première case d’un athlète lors de l’enregistrement des résultats.
Nombre de couloirs
Si la valeur indiquée ici est plus élevée que le nombre d’athlètes par série, les couloirs surnuméraires sont imprimés vides.
S’il y a un tour de compétition supplémentaire, il est possible de mettre les paramètres de qualification. Ceux-ci sont imprimés en haut sur les feuilles de concours.
Vainqueur de série
Le nombre des meilleurs par série (vainqueur de série, deuxième de série, etc.), qui se sont qualifiés pour le tour suivant.
Meilleures performances
Athlètes supplémentaires qui sont qualifiés au vu de la performance réalisée.
Total
Le nombre total des athlètes à qualifier pour le tour suivant est calculé automatiquement à l’enregistrement.
