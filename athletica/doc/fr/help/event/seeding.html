Organiser les séries

L’organisation des séries peut être influencée par les paramètres suivants:

Nombre d‘athlètes

Cette valeur définit combien d’athlètes sont au maximum mis par série.

Nombre de couloirs

Si cette valeur est supérieure au nombre d‘athlètes les couloirs surnuméraires sont mis vides. Ceci est utile si tous les couloirs disponibles ne doivent pas être occupés, mais que les couloirs surnuméraires doivent tout de même être affichés et imprimés sur les formulaires.

Ce paramètre ne peut être appliqué que pour les courses.

Mode de répartition

Le mode de répartition définit comment les athlètes sont répartis dans les différentes séries:
ouvert
Les athlètes sont répartis dans les séries et les couloirs en fonction du principe du hasard.
Meilleures performances ensemble
Si les meilleures performances ont été enregistrées lors des inscriptions, cette option permet aux meilleurs athlètes de s’aligner dans la même série.
Meilleures performances réparties
Si les meilleures performances ont été enregistrées lors des inscriptions, cette option permet de répartir les meilleurs athlètes dans toutes les séries.
Équipe
Les athlètes sont répartis dans les séries et les couloirs selon l’équipe.
