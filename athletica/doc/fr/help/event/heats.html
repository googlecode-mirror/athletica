Séries

Ici on procède à l’organisation des séries pour toutes les disciplines. Non seulement les disciplines de course, mais aussi les disciplines techniques peuvent être organisées en séries.

Sélection des tours

Si la fonction est appelée à partir du moniteur, le tour correspondant de l’épreuve est déjà sélectionné. Si la fonction est appelée par le menu, il faut d’abord sélectionner la catégorie, l’épreuve et le tour souhaités.

En fonction du statut, le formulaire est un peu différent. Si des résultats sont déjà enregistrés, un avertissement approprié est affiché. Au moyen du bouton Ouvrir aperçu des résultats il est possible dans ce cas d’activer directement la fonction correspondante.

Organisation des séries

Si aucune série n’est encore organisée ou si l’organisation a été effacée, ce bouton ouvre le dialogue vers Organisation des séries, resp. vers Qualification.

Pour les finales des courses les meilleurs athlètes sont automatiquement répartis dans la dernière série. Les séries sont désignées par A, B, C etc. (A pour la série la plus rapide).

Pour les concours multiples les athlètes sont répartis selon le nombre de points des disciplines précédentes pour disputer la dernière discipline de course ensemble.

Ajouter nouveaux athlètes/relais

Si la liste de départ contient d’autres athlètes ou relais qui ne sont pas répartis dans une série, une fonction appropriée est à disposition pour les répartir. Ils peuvent soit être répartis dans une série existante soit dans une nouvelle série. Les nouvelles séries reçoivent automatiquement l’appellation '0'.
Ici apparaissent tous les athlètes ou relais qui, à l’appel, ont été notés comme pas présents, qui ont été supprimés de la répartition dans les séries ou qui n’ont pas réussi à se qualifier pendant le tour précédent.

Fonctions générales

Si ce tour de compétition comprend plus qu’une série, des boutons de commande avec les noms des séries sont affichés pour pouvoir passer directement à la série souhaitée dans le formulaire.
Aussi longtemps que les séries sont en traitement, un avertissement correspondant est affiché. Les fonctions générales suivantes sont à disposition:

Attribuer l’installation à toutes les séries

Parmi les installations du stade, la même peut être attribuée à toutes les séries, par ex. Ligne droite d‘arrivée pour les sprints.
Cette information n’est pas impérativement nécessaire pour la suite du déroulement. Elle n’est imprimée que pour information sur les feuilles de concours.

Effacer des séries

La répartition dans les séries est effacée et peut être refaite à nouveau.

Terminer répartition

Avec cette fonction, l’utilisateur peut terminer la répartition dans les séries. Le statut de ce tour de compétition est modifié en fonction et la Fonction d‘impression est appelée.

Imprimer

Après avoir terminé la répartition dans les séries, la Fonction d‘impression peut être appelée directement.

Liste des séries

Appellation des séries

L’appellation attribuée automatiquement à la série (numérotation croissante) peut être modifiée manuellement. Cela ne doit pas être obligatoirement un numéro, les lettres font aussi l’affaire. Une appellation peut au maximum comporter deux éléments et ne doit apparaître qu’une fois par tour.

Installation

Ici on peut attribuer une installation spéciale à une certaine série, par ex. longueur 1 à la première de saut en longueur, longueur 2 à la deuxième.
Cette information n’est pas impérativement nécessaire pour la suite. Elle n’est imprimée que pour information sur les feuilles de concours.

Liste de départ

Modifier la répartition

Les répartitions dans les séries ou les positions attribuées automatiquement peuvent être modifiées manuellement, même après la clôture de la répartition. Pour cela il faut cliquer sur l’athlète ou le relais pour le sélectionner. Un deuxième clic de souris sur le couloir souhaité déclenche la modification. En cliquant à nouveau il est possible de supprimer une sélection.

Une position peut être attribuée plusieurs fois. Pour les courses les couloirs peuvent ainsi être occupés à double. Pour les autres disciplines, la position sert uniquement à déterminer l’ordre de passage.

Supprimer athlète/relais

Un athlète ou un relais peut être supprimé d’une liste de départ, en cliquant sur la ligne correspondante pour la sélectionner. En pressant le bouton de commande Supprimer on enlève l’inscription de la répartition dans les séries. Une nouvelle répartition ultérieure est possible.

Nouvelle série

Pour créer une nouvelle série, un athlète ou un relais doit être sélectionné par un clic et être placé à la fin de la liste de départ sur la place de start prévue à cet effet. La nouvelle série reçoit l’appellation '0', apparaît donc tout en haut et doit être adaptée manuellement.
