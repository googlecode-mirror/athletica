Chronométrage

Athletica est compatible avec le chronométrage Omega (Scan-o-vision) et Alge (OPTIc et OPTIc2).

Si aucun échange de résultats de chronométrage n’est souhaité, il faut sélectionner "aucun de chronométrage".

Classer automatiquement

Après que les résultats de tous les athlètes aient été transférés, le classement est automatiquement terminé, dans la mesure où la coche pour "classer automatiquement " est mise.

Données

chemin
Ici il faut définir le cheminement par lequel les fichiers d’échange de données entre chronométrage et Athletica sont enregistrés .

Sponsor
Pour le chronométrage Omega, un texte peut être défini dans le champ sponsor. Celui-ci sera ensuite par ex. reporté sur le Score-Board (si disponible).
