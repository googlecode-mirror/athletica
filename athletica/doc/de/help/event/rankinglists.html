Ranglisten

Mit dieser Funktion können Gesamtranglisten, Ranglisten pro Kategorie, pro Wettkampf oder pro Wettkampfrunde gedruckt werden. Je nach Auswahl stehen verschiedene Druckoptionen zur Verfügung.

Typ

Die folgenden Listen können generiert werden:
Einzel
Einzelranglisten, eventuell eingeschränkt auf eine Kategorie und/oder einen Wettkampf.
Mehrkampf
Mehrkampfranglisten, eventuell eingeschränkt auf eine Kategorie. Pro Athlet werden die Zählresultate pro Disziplin gezeigt.
Mannschaftswettkämpfe
Ranglisten für Mannschaftswettkämpfe, z.B. SVM.
Vereinsrangliste
Rangliste der Teams nach Totalpunktzahl pro Kategorie, eventuell eingeschränkt auf eine Kategorie. Beim normalen Vereinswettkampft werden pro Team die eingerechneten Disziplinen und je die Anzahl der gewerteten Athleten gezeigt. Bei Vereins-Teammehrkämpfen werden alle gewerteten Mehrkämpfer aufgeführt.
Notenblätter
Formular mit allen Zählresultaten pro Team zur Unterzeichnung durch die Schiedsrichter und Mannschaftsführer. Die Auswahl kann auf eine einzelne Kategorie eingeschränkt werden.

Seitenumbruch

Je nach Ranglistenauswahl kann nach jeder Kategorie oder nach jeder Disziplin ein Seitenumbruch eingefügt werden.

Optionen

Titelblatt
Ein Titelblatt mit Meetingbezeichnung und Angaben zur Organisation wird vorangehängt.

Funktionen

Drucken
Ein neues Browserfenster wird geöffnet und die Druckfunktion gestartet.
Anzeigen
Die Rangliste wird im selben Browserfenster dargestellt.
