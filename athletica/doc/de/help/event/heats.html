Serien

Hier wird die Serieneinteilung für alle Disziplinen vorgenommen. Nicht nur Laufdisziplinen, auch technische Disziplinen können in Serien eingeteilt werden.

Rundenauswahl

Wird die Funktion vom Monitor aus aufgerufen, ist die entsprechende Wettkampfrunde bereits ausgewählt. Wird die Funktion über das Menu gestartet, muss zuerst die gewünschte Kategorie, Wettkampf und Runde ausgewählt werden.

Je nach Status präsentiert sieht das Formular etwas anders aus. Sind bereits Resultate erfasst, wird eine entsprechende Warnung gezeigt. Mit der Schaltfläche Resultatansicht öffnen kann in diesem Fall direkt die entsprechenden Funktion aktiviert werden.

Serieneinteilung

Sind noch keine Serien eingeteilt oder wurde die Einteilung wieder gelöscht wird mit dieser Schaltfläche der Dialog zur Serieneinteilung, bzw. zur Qualifikation geöffnet.

Neue Athleten/Staffel hinzufügen

Falls sich auf der Startliste weitere Athleten oder Staffeln befinden, welche nicht einer Serie zugeteilt sind, steht eine entsprechende Funktion zur Verfügung, um diese einzuteilen. Diese können einer bereits bestehenden Serie oder einer neuen Serie zugeteilt werden. Neue Serien erhalten automatisch die Bezeichnung '0'.
Hier erscheinen alle Athleten oder Staffeln, welche beim Appell als nicht anwesend bezeichnet wurden, welche aus der Serieneinteilung gelöscht wurden oder welche sich aus einer vorhergehenden Runde nicht qualifizieren konnten.

Allgemeine Funktionen

Falls diese Wettkampfrunde mehr als eine Serie umfasst, werden Schaltflächen mit den Seriebezeichnungen gezeigt, um innerhalb des Formulars direkt zur gewünschte Serie zu springen.
Solange die Serien noch in Bearbeitung sind wird eine entsprechende Warnung gezeigt. Die folgenden allgemeinen Funktionen stehen zur Verfügung:

Anlage allen Serien zuweisen

Aus der Liste der Stadionanlagen kann allen Serien dieselbe Anlage zugewiesen werden, z.B. Zielgerade bei Sprints.
Diese Angabe ist für den weiteren Ablauf nicht zwingend nötig. Sie wird nur zur Information auf die Wettkampfblätter gedruckt.

Serien löschen

Die Serieneinteilung wird gelöscht und kann nachher wieder neu vorgenommen werden.

Einteilung abschliessen

Der Benützer kann mit dieser Funktion die Serieneinteilung abschliessen. Der Status dieser Wettkampfrunde wird entsprechend geändert und die Druckfunktion aufgerufen.

Drucken

Nach Abschluss der Serieneinteilung kann die Druckfunktion direkt aufgerufen werden.

Liste der Serien

Seriebezeichnung

Die automatisch vergebene Seriebezeichnung (aufsteigende Nummerierung) kann manuell geändert werden. Dies muss nicht zwingend eine Nummer sein, ebensogut können Buchstaben vergeben werden. Eine Bezeichnung darf maximal zweistellig sein und pro Runde nur einmal vorkommen.

Anlage

Hier kann einer einzelnen Serie eine spezielle Anlage zugwiesen werden, z.B. Weit 1 für die erste Weitsprungserie, Weit 2 für die zweite.
Diese Angabe ist für den weiteren Ablauf nicht zwingend nötig. Sie wird nur zur Information auf die Wettkampfblätter gedruckt.

Startliste

Einteilung ändern

Die automatisch vergebenen Serie- und Positionseinteilungen können manuell auch nach dem Abschliessen der Einteilung geändert werden. Dazu muss der Athlet oder die Staffel durch anklicken selektiert werden. Ein zweiter Mausklick auf die gewünscht neue Bahn löst die Änderung aus. Eine Selektion kann durch nochmaliges anklicken wieder aufgehoben werden.

Eine Position kann mehrmals vergeben werden. Damit können bei Läufen Bahnen doppelt belegt werden. Bei den andern Disziplinen dient die Position einzig zur Bestimmung der Startreihenfolge.

Athlet/Staffel Löschen

Ein Athlet oder eine Staffel kann aus der Startliste gelöscht werden, indem die entsprechende Zeile durch anklicken selektiert wird. Durch Drücken der Schaltfläche Löschen wird der Eintrag aus der Serieneinteilung entfernt. Eine spätere Wiedereinteilung ist möglich.

Neue Serie

Um eine neue Serie zu erstellen, muss ein Athlet oder eine Staffel durch Anklicken selektiert und am Ende der Startliste auf den dafür vorgesehenen Startplatz gesetzt werden. Die neue Serie erhält die Bezeichnung '0', erscheint damit zuoberst und muss manuell angepasst werden.
