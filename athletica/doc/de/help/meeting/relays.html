Staffeln

Die Liste links zeigt alle angemeldeten Staffeln. Die Liste kann nach Kategorie, Disziplin oder Staffelname sortiert werden. Durch einen Mausklick auf eine Staffel werden deren Details im Hauptteil des Fensters angezeigt.

Befehle

Drucken

Im Hauptteil wird ein Formular zum Drucken von Staffelanmeldungen angezeigt.

Neue Anmeldung

Aufruf einer Eingabemaske zur Erfassung einzelner Anmeldungen.

Staffeldaten ändern

Nach Anklicken einer Staffel aus der Liste wird im Hauptteil ein Formular zur Änderung der Staffeldaten geöffnet. Der Staffelname ist unabhängig von Verein oder Mannschaft und kann dementsprechend frei gewählt werden.

Eine Mannschaft kann nur zugeteilt werden, sofern diese Kategorie als Mannschaftswettbewerb (z.B. Vereinsmeisterschaft) durchgeführt wird und die Mannschaft bereits erfasst wurde.

Bei der Staffelanmeldung kann eine Bestleistung gesetzt werden, welche nachher aber aufgrund der im Wettkampf erzielten Resultate überschrieben wird. Diese Zeit kann bei der Serieneinteilung verwendet werden.

Neue Athleten können aus den untenstehenden Listen ausgewählt werden. Nach der Auswahl eines Athleten kann dieser durch Vergabe einer Startposition der Staffel zugeteilt werden. Die Startposition sind aufsteigend nummeriert, wobei eine Position nur einmal vergeben werden kann.
Um einen Athleten aus der Staffel zu entfernen genügt es, seine Startposition zu löschen oder auf Null zu setzen.

nicht zugeteilte Athleten

Die Liste der nicht zugeteilten Athleten umfasst jene Teilnehmer, die für die gewählte Kategorie, den gewählten Verein und allenfalls die gewählte Mannschaft in dieser Staffeldisziplin gemeldet sind. Die Athleten aus dieser Liste können direkt der Staffel zugeteilt werden.

andere Vereinsathleten

Je nach Meeting kann es hilfreich sein, andere Athleten einer Staffel zuzuteilen. Die Liste unten rechts zeigt dazu alle weiteren Athleten desselben Vereins. Auch diese können so direkt der Staffel zugeteilt werden.

Anmeldung löschen

Um eine Staffel zu löschen muss diese zuerst per Mausklick selektiert werden. Danach kann sie mit dem Schalter Löschen gelöscht werden. Dies ist nur möglich, wenn die Staffel nicht bereits in einer Wettkampfserie eingeteilt ist.
