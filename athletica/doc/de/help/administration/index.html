Administration

Über dieses Hauptmenu können verschiedenen Untermenus zur Administration der Stammdaten aufgerufen werden. Auf der Hauptmenu-Seite befinden sich Funktionen zur Konfiguration von Applikations-Parametern sowie verschiedene weiterführende Links.

Konfiguration

Die Applikationssprache kann hier gesetzt werden. Sie wird beim einzelnen Benützer als Cookie gespeichert.

Links

Unter diesem Titel finden sich Verweise zur Dokumentation sowie zu den Lizenzbestimmungen.

Datenbank

Daten sichern

Beim Sichern werden die Daten in einer SQL-Datei auf dem PC des Benützers abgelegt. Der Benützer kann Name und Ort der Ablage frei wählen.

Sicherung einspielen

Zum Einspielen einer Datensicherung muss der Benützer zuerst die entsprechende SQL-Datei auswählen. Entspricht die Version der gesicherten Daten der installierten Version, wird die Datei eingelesen.

Achtung: Bestehende Daten werden dabei gelöscht! Vor dem Einspielen also nochmals eine Sicherung der bestehenden Daten vornehmen.

Cache leeren

Mit dieser Funktion können Probleme mit nicht gefundenen Athleten und/oder Vereine behoben werden.

Achtung: erfasste Athleten und Vereine werden gelöscht, damit diese neu vom Stammdatenupdate heruntergeladen werden können.
Dies setzt voraus, dass für die in Athletica erfassten Meetings noch keine Athleten erfasst wurden.
