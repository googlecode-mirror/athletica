Stadien

Bedienung

Ein neues Stadion wird mit dem Formular am oberen Seitenende aufgesetzt und muss gespeichert werden.
Änderungen an bestehenden Daten werden direkt in der Datenbank gespeichert, wenn zum nächsten Feld gewechselt wird.
Ein Stadion kann nur gelöscht werden, wenn er nicht mehr referenziert wird.

Daten

Stadion
Name des Stadion, wird jeweils alphabetisch geordnet.
Bahnen
Anzahl Bahnen, wird als Standardwert verwendet.

Anlagen

Pro Stadion können die verschiedenen Wettkampfanlagen definiert werden.
